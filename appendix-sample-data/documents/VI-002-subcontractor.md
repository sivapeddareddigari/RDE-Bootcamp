# Notes for Analyst

- Maps to TX-2026-04-0044 (USD 2,400 cost).
- This is a **subcontractor**, not a vendor. Per contract, 8% markup applies on subcontractor pass-through.
- Therefore the line on the invoice to Northstar should be USD 2,400 + 8% = USD 2,592.
- The transaction record contains the cost only. The markup line is the Analyst's to add (or someone's) on the invoice.
- This is the kind of step that gets forgotten when an Analyst is rushing.
